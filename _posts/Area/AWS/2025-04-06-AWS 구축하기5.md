# 9.2 S3
## 9.2.1 S3의 작업 범위와 비용
- S3는 스토리지 관리를 수행하는 매니지드 서비스
- S3는 EBS 보다 우수함

## 9.2.2 S3와 VPC의 관계
- S3는 VPC밖에 생성
- S3 접근 방법
    - 인터넷으로부터 직접 접근
    - VPC로부터 접근

## 9.2.3 S3 시스템
- S3 데이터 저장을 위해 버킷(bucket)을 생성해야함
- 버킷 안에는 서비스에서 이용하는 객체인 object를 저장해야한다
    - 객체는 텍스트, 이미지, 음성, 동영상 등을 취급
